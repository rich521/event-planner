#####Event planner

Login, Sign in, Add Event

#####Instructions:

1. Open index.html to run web application

#### Gulp instructions 

To run on gulp and browser sync

1. Open terminal
2. cd "/to your project folder"
3. run "-npm install" 
4. npm install --global gulp-cli (if you have a previous version of gulp installed)
5. run "gulp"
